Plan‑Trek Travel Guide Console
This page reads a planet travel‑guide JSON file and renders a Tec‑Trek‑style console UI: left list + right detail pane. Place Pla‑Build‑0.json in the same folder as this HTML file (or edit the DATA_URLS array in the script).
Intended content: location, appearance, geography, cities, inhabitants, tourism, and weather—only for worlds with enough canon detail to support a real “travel entry.”
